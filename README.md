# Independence Quest v2

A grounded, Bob-flavored life-skills RPG for building independence through quests, subquests, bosses, budgeting, and chapter progression.

## Play

Once GitHub Pages is enabled, the game should be available at:

<https://kerouacwannabe.github.io/Independence-Quest-v2/>

## What’s included

- Real quest progression with subquests and auto-completion
- Character classes with different mechanics
- Random boss encounters by chapter
- Budgeting and housing examples
- Save slots, export/import, family support mode
- Desktop/mobile responsive static web app

## Source

The editable source now lives in:

- `independence-quest/` — Vite + TypeScript app source
- `docs/` — built Pages output

## Publishing

This repo is set up to build `independence-quest/` and deploy the generated `docs/` site to GitHub Pages via GitHub Actions.

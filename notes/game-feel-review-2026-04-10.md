# Independence Quest v2 Game-Feel Review

Date: 2026-04-10

## Blunt findings
- The app is strongest when it tells the user exactly what to do next.
- It gets noisy when too many supporting cards compete with that next action.
- Class identity is helpful, but some class systems still read as flavor instead of a real gameplay choice.
- Bosses are better when they feel like special events instead of larger checklists.
- The onboarding flow should get to the first move faster.

## UI weirdness
- Some screens still carry more explanatory text than they need.
- Progress information is spread across too many places.
- The app needs stronger visual hierarchy for first-session users.

## Prioritized follow-ups
1. Keep collapsing Today toward one primary action and one fallback.
2. Make bosses feel eventful and different.
3. Make class powers and progression visibly change the game state.
4. Reduce player-facing copy where it does not improve action.
